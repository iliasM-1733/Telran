метод, который принимает строку - название фигуры "треугольник", "круг", "прямоугольник"
согласно тому, какая фигура пришла, вызвать соответствующий метод для подсчета площади фигуры.
Использовать if-else
дополнительные параметры (размеры сторон) запросить непосредственно в каждом отдельном методе
для подсчета площади треугольника использовать формулу Герона:
S = sqrt(p * (p - a) * (p - b) * (p - c)), где sqrt - корень квадратный, p - полупериметр (периметр / 2).
для площади круга использовать статическую переменную класса Math

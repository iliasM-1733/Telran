Задача:
Написать программу, которая будет проверять, является ли введенная строка палиндромом.
Палиндром - это строка, которая читается одинаково как слева направо, так и справа налево.

Примеры палиндромов:

"level"
"racecar"
"madam"
"A man, a plan, a canal: Panama"
Подсказка: можно удалить из строки все знаки пунктуации и привести все символы к одному регистру, чтобы убедиться, что строка является палиндромом.

for() от  0 до середины.
charAt()
Если при сравнении символы не равны, то return false;
Return true после цикла for




public static boolean isPalindrome(String str) {
    if (str == null) { // проверяем, что строка не пустая
        return false;
    }
    str = str.toLowerCase(); // приводим все символы строки к нижнему регистру
    for (int i = 0; i < str.length() / 2; i++) { // перебираем символы строки до середины
        if (str.charAt(i) != str.charAt(str.length() - i - 1)) { // сравниваем символы с конца и начала
            return false;
        }
    }
    return true;
}

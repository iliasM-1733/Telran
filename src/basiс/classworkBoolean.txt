написать приложение (отдельный метод) areWeGoingToExcursion
который принимает 3 аргумента -
- число людей в группе int
- есть ли экскурсовод boolean
- хорошая ли погода? boolean

если набралось больше 5 человек, то условие по количеству людей выполняется
если выполняются 2 из трех условий, то мы идем на экскурсию.

метод имеет тип возвращаемого значения boolean

1. создать булевую переменную для группы
        операторы сравнения
2. попарно сравнить, для этого создать две переменные
        операторы && ||
   b1 - набралась группа * экскурсовод
   b2 - набралась группа * хорошая ли погода
   b3 -  хорошая ли погода  * экскурсовод
3. сравнить результаты попарного сравнения
        boolean result = b1 # b2 # b3
